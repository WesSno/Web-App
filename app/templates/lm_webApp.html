Borehole Analysis Web App

Pumping Test is a long and boring process coupled with difficult calculations to analyse the data obatined from the test. This can make carrying out the test in the right procedure quite a challenge as it takes up too much time.

Therefore, this web application aims to shorten the time involved by making the analysis stage as simple as possible. All that is needed is data from the pumping test and the analysis stage is made easier and simple.
